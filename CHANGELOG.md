### ✨ v0.7.18 更新日志

- 🛡️ **新增强兼模式：** 在桌面端弹窗的“深度兼容模式”中新增可选的“强兼模式”，用于对付会反复篡改 DOM 或样式、试图遮挡 PageDye 背景的顽固网站；开启后会更高频地检测 DOM/style/class 变化并恢复 PageDye 覆盖。
- 🔧 **增强深度兼容抗回滚能力：** 强兼模式会在开启后的短时间内高频压制，持续重建被删除的 PageDye 根节点/样式，反复重写已处理遮挡层，并额外压制大面积遮挡元素的 `::before` / `::after` 伪元素背景。
- 🎛️ **优化弹窗布局与滚动体验：** 收紧手风琴标题高度，让设置项更紧凑；状态栏改为随内容滚动；底部导航不再被滚动条遮挡；中间内容区滚动条默认隐藏但仍可滚动。
- 🧩 **优化强兼模式可用状态：** 未开启“为此网站启用”时，强兼模式整组会显示为灰色禁用状态，避免误触。

### ✨ v0.7.17 更新日志

- 🛡️ **受浏览器安全限制的页面现在会明确禁用：** 当 PageDye 无法在当前标签页运行时（例如 Chrome 受保护页面、Chrome Web Store、未授权的本地文件页等），弹窗会自动切换为灰色禁用状态，不再让用户误以为设置已经生效。
- 🧭 **新增居中的安全限制提示层：** 在无法运行的页面中，弹窗会显示“由于浏览器安全限制无法运行”的提示卡片；提示层固定在弹窗可视区域中心，并锁定滚动，避免滚动时提示跑偏。
- 🧪 **新增严格 CSP 本地测试站点：** 添加 `test-sites/strict-csp-hostile`，用于手动测试严格 CSP、禁止 iframe、厚背景层和主动移除注入节点等极端兼容场景。

### ✨ v0.7.14 更新日志

- 🎨 **动效可以叠加在背景上了：** "背景类型"重新设计为"背景（无 / 纯色 / 渐变 / 图片，四选一）+ 动效独立开关"的组合模式——现在可以在纯色、渐变或图片背景上叠加一层动画 Canvas 壁纸，也可以像以前一样单独把动效当背景用；矩阵雨、雨滴等带拖尾效果的动效叠加时会正确透出下层背景，不再整片糊成纯色。
- 🖱️ **新增自定义光标：** 可在任意网站启用自定义光标，提供实心球、空心环、发光球、点环组合四种样式，颜色、大小、悬停放大倍数均可自定义；默认精确 1:1 跟随鼠标，也可以开启"平滑跟随"体验缓动效果，并支持渐隐 / 彗星 / 闪烁三种鼠标拖尾。
- 🧊 **磨砂玻璃支持自定义底色：** 每个磨砂玻璃元素都可以单独设置底色和透明度，不再只能使用固定的白/黑半透明遮罩；适合给卡片、侧栏、搜索框做更贴合壁纸的玻璃拟态效果。
- 🧭 **弹窗与设置页编辑器同步优化：** 背景选择改成更直观的"无 / 纯色 / 渐变 / 图片 / 动效"入口，动效叠加状态会在预览卡片上显示小圆点提示；旧版保存的"动效背景"会自动迁移为"无背景 + 动效叠加"，原有配置不会丢。
- 🎛️ **设置页视觉升级：** 选项页补齐与弹窗一致的背景编辑控件、颜色预设、模式切换和光标设置，并整体调整为更统一的 Material Design 3 风格，浅色/深色主题的色彩、按钮、卡片、表格和表单状态都更协调。
- 🛠️ **修复 PageDye Lite 底部导航切换闪烁问题并优化渲染性能：** 之前切换底部标签页（壁纸 / 磨砂玻璃 / 高级设置）时会重建整个控制面板，导致导航栏跟着闪烁；现在只更新变化的标签页内容区域，并清理重复残留的淡入动画，低性能设备上切换和编辑设置更跟手。
- 🧹 **下线官网体验版小组件：** 移除了官网上的"体验版"网页小组件，避免移动端访问时因为在线演示过重而卡顿；下载与功能说明保留在官网和 Releases 中。

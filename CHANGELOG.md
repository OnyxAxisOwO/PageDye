- 🖱️ **新增自定义光标：** 可在任意网站启用自定义光标，提供实心球、空心环、发光球、点环组合四种样式，颜色、大小、悬停放大倍数均可自定义；默认精确 1:1 跟随鼠标，也可以开启"平滑跟随"体验缓动效果，并支持渐隐 / 彗星 / 闪烁三种鼠标拖尾。
- 🧊 **磨砂玻璃支持自定义底色：** 每个磨砂玻璃元素都可以单独设置底色和透明度，不再只能使用固定的白/黑半透明遮罩；适合给卡片、侧栏、搜索框做更贴合壁纸的玻璃拟态效果。
- 🧭 **弹窗与设置页编辑器同步优化：** 背景选择改成更直观的"无 / 纯色 / 渐变 / 图片 / 动效"入口，动效叠加状态会在预览卡片上显示小圆点提示；旧版保存的"动效背景"会自动迁移为"无背景 + 动效叠加"，原有配置不会丢。
- 🎛️ **设置页视觉升级：** 选项页补齐与弹窗一致的背景编辑控件、颜色预设、模式切换和光标设置，并整体调整为更统一的 Material Design 3 风格，浅色/深色主题的色彩、按钮、卡片、表格和表单状态都更协调。
- 🛠️ **修复 PageDye Lite 底部导航切换闪烁问题并优化渲染性能：** 之前切换底部标签页（壁纸 / 磨砂玻璃 / 高级设置）时会重建整个控制面板，导致导航栏跟着闪烁；现在只更新变化的标签页内容区域，并清理重复残留的淡入动画，低性能设备上切换和编辑设置更跟手。
- 🧹 **下线官网体验版小组件：** 移除了官网上的"体验版"网页小组件，避免移动端访问时因为在线演示过重而卡顿；下载与功能说明保留在官网和 Releases 中。

### ✨ v0.7.13 更新日志

- 🎨 **纯色预设焕新：** 浅色、深色预设各自独立成组，一共 28 种颜色，深色系不用切换系统主题也能选到耐看的深色背景；点击"+"可以把当前颜色存成自定义快捷色块，随时可移除。
- 🌈 **重做全部 12 款渐变风格：** 换成了日暮、深渊、翡翠森林、北极光等更协调耐看的配色，替换掉了偏跳脱花哨的旧配色。
- 🛠️ **修复弹出窗口背景类型切换 bug：** 选择"图片"或"动效"背景类型后，下方的具体设置项（特效样式、密度、速度、上传等）不显示的问题。
- 🛠️ **修复控制面板深色模式下文字不可见问题：** 在"外观"页自定义了浅色背景、但系统/浏览器处于深色模式时，"已配置网站"列表里的域名和背景类型标签会显示成看不见的白字叠白底。
- 🛠️ **修复颜色输入框字体问题：** 部分系统下颜色值输入框会错误显示为宋体，现在统一使用等宽字体。
- 🛠️ **修复时段模式类型标签空白问题：** 使用"自定义时段"模式的网站，在"已配置网站"列表里的类型标签之前会显示空白，现在会正确显示"时段 (N)"及对应的多色预览图标了。

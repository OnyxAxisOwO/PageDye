功能

多种背景类型

纯色、线性/径向渐变、图片，或 16 种动态特效壁纸（Matrix 雨、粒子、极光、雪花、打字机等）

磨砂玻璃

为指定卡片/容器元素加毛玻璃背景，透出下层壁纸，模糊度与底色独立可调

昼夜双态联动

跟随系统 prefers-color-scheme 自动切换浅色/深色壁纸

幻灯轮换

多张壁纸按间隔自动轮换，支持随机顺序

背景选择器

网站自身 CSS 遮挡整页背景时，直接把背景应用到任意元素

完全本地

所有设置存储在浏览器本地，不联网、不追踪，支持导出/导入备份

自定义动效 API v0.6

在设置页里编写自己的 Canvas 动效，实时预览，和内置的 16 种一样直接在任意站点选用。仅扩展版支持。

桌面安装

扩展尚未上架各浏览器商店，目前需要手动加载已解压的扩展包。

前往 GitHub Releases 下载最新版

Chrome / Edge / Brave 等 Chromium 内核浏览器

  1. 解压下载的压缩包
  2. 地址栏输入 chrome://extensions（Edge 为 edge://extensions）
  3. 打开右上角的开发者模式
  4. 点击加载已解压的扩展程序，选择解压后的文件夹

Firefox（需 140+；Firefox for Android 需 142+）

  1. 地址栏输入 about:debugging#/runtime/this-firefox
  2. 点击临时载入附加组件…
  3. 选择解压后文件夹内的 manifest.json

扩展未经 Mozilla 签名，只能以"临时扩展"方式加载，浏览器重启后需要重新加载一次。

手机 / 平板：PageDye Lite

精简的用户脚本（UserScript）版本，核心渲染逻辑和扩展版共享，功能是完整版的子集，通过脚本管理器运行。

已经装好 Tampermonkey / Violentmonkey / Userscripts 的话，直接点这个：

一键安装 PageDye Lite

点击后脚本管理器会自动弹出安装确认页面，确认即可，不需要手动复制粘贴代码。

Android — Edge / Firefox

  1. 在浏览器的扩展 / 附加组件商店搜索并安装 Tampermonkey
  2. 回到本页，点上面的一键安装按钮

Chrome 移动版官方不支持任何扩展，装不了 Tampermonkey；可以换用 Edge、Firefox，或 Kiwi Browser 等支持扩展的第三方 Chromium 分支。

iOS / iPadOS — Safari

  1. App Store 搜索安装 Userscripts（作者 Quoid，免费开源）
  2. 设置 → Safari → 扩展 → 启用 Userscripts，并允许在所有网站上运行
  3. 回到本页，点上面的一键安装按钮

安装完成后，打开任意网站，右下角会出现一个悬浮按钮，点开即为设置面板；按钮的颜色、大小、图标、拖动、贴边隐藏都可以在面板的"高级设置"标签页里自定义。
